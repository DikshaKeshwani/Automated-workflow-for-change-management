# Automated-workflow-for-change-management

A web application designed to streamline the process of creating, reviewing, approving, and tracking change requests in an organization. This system ensures a systematic and efficient approach to managing change workflows.

---

## Features
- **Dashboard**: View all change requests with their current status.
- **Create Change Requests**: Submit new change requests with title, description, and creator information.
- **Review Changes**: Detailed view of each change request for assessment.
- **Approval Workflow**: Approve pending change requests to move them forward.
- **Integrated System**: Easy-to-use interface with automated workflows.

---

## Technology Stack
- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite
- **Testing**: Unittest
- **Deployment**: Flask development server (can be deployed on Heroku, AWS, etc.)
